Features

  • Clear cut MVC architecture
  • OOP code
  • Hooks to extend features
  • ORM to manage database
  • URL rewriting
  • Translatable using Gettext
  • Composer-friendly

Requirements

  • A webserver (Apache and Nginx are fully supported)
  • PHP 5.3.0 or later
  • A database such as MySQL (MariaDB, PostgreSQL and SQLite are also supported)
